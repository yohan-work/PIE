🚀 자동화 워크플로우

📝

1. Git Commit

코드 변경사항을 GitHub에 푸시하면 자동으로 파이프라인이 시작됩니다.

⚙️

2. GitHub Actions

코드 품질 검사, 린팅, 포맷팅 검증을 자동으로 수행합니다.

🚀

3. 자동 배포

Vercel을 통해 미리보기 또는 프로덕션 환경에 자동 배포됩니다.

📸

4. 스냅샷 테스트

Playwright로 UI 변경사항을 자동 감지하고 스크린샷을 비교합니다.

♿

5. 접근성 검사

axe-core와 Pa11y로 WCAG 가이드라인 준수를 자동 검증합니다.

📊

6. 리포트 생성

모든 테스트 결과를 종합하여 시각적 리포트를 자동 생성합니다.
